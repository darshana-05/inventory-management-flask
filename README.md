# Inventory Management Web Application

## Description
This is a simple Inventory Management web application built using Python Flask and SQLite. 
The application allows users to manage products, warehouse locations, and track product 
movements between locations. It also provides a balance report showing product quantities 
available in each warehouse.

## Features
- Add and view products
- Add and view warehouse locations
- Record product movements (inward, outward, and transfers)
- Generate inventory balance report per product and location

## Technologies Used
- Python
- Flask
- SQLite
- HTML (basic)
